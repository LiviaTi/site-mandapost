Política de Privacidade - MandaPost

Última atualização: 22 de Fevereiro de 2026.

Esta política estabelece como o MandaPost (doravante "Plataforma") trata dados pessoais em conformidade com a Lei Geral de Proteção de Dados (LGPD - Lei 13.709/2018).

1. Agentes de Tratamento e Coleta

A Plataforma coleta dados estritamente necessários para a execução do serviço de marketing digital via WhatsApp, incluindo:

  • Dados Identificadores: Nome, número de telefone vinculado ao WhatsApp e e-mail.
  • Conteúdo de Mídia: Fotos, vídeos e textos enviados pelo Usuário para processamento por IA.
  • Tokens de Acesso: Credenciais de autorização (OAuth) para Instagram e Facebook via API Oficial.

2. Processamento por Inteligência Artificial (IA)

O Usuário declara ciência de que as mídias e textos enviados serão processados por modelos de Inteligência Artificial de terceiros (como OpenAI, Anthropic ou Google Gemini). A Plataforma garante que este processamento visa exclusivamente a prestação do serviço contratado, não havendo comercialização de dados pessoais para treinamento de modelos públicos.

3. Compartilhamento com Terceiros

Para o funcionamento do serviço, os dados são compartilhados com:

  • Meta Platforms, Inc.: Para o envio de mensagens (WhatsApp) e publicações (Instagram/Facebook).
  • Provedores de Infraestrutura: Serviços de nuvem (Supabase/AWS) com criptografia de ponta.

4. Direitos do Titular

O Usuário poderá, a qualquer tempo, solicitar a exclusão de seus dados ou revogar o acesso às suas contas sociais através do canal oficial de suporte via WhatsApp.
